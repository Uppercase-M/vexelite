## Description
Vexelite is a simple voxel game like Minecraft, made entirely in [Turbowarp](https://turbowarp.org/) which is a [Scratch](https://scratch.mit.edu/) mod. Powered by [Simple3D](https://extensions.turbowarp.org/Xeltalliv/simple3D/).
Please keep in mind that this is a very early verion and things are going to change.

## Installation
To play the game, simply download the *.sb3* file and once you've opended it, click the green flag.
